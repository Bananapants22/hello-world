# hello-world
A repository to store how to say/output "Hello, World!" in as many languages as possible.

## Supported Languages
* **Assembly** (on _**Raspberry Pi**_)
* **Brainf***** (sometimes referred to as _**branflakes**_)
* **C**
* **Emoji**
* **English**
* **Fortran**
* **French**
* **German**
* **H9+**
* **HQ9+**
* **HQ9+B**
* **HQ9++**
* **Java**
* **Ook**
* **Python**
* **Ruby**
